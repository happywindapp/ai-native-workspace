## Assembly Rules

- TOC is mandatory and links to every `##`/`###` heading.
- Terminal commands and Mermaid code go in fenced code blocks.
- Prose: professional, concise. Do not narrate the analysis process.
- If the file already exists, update sections in place; preserve unrelated content; ask before destructive overwrite.
- Each part can be generated by a separate subagent then merged, for large workspaces.
